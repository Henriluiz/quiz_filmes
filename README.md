# 🎬 Quiz de Filmes  

Um site de **Quiz sobre Filmes** desenvolvido com **PHP** e **CSS**, rodando em servidor local com **XAMPP**.  
O projeto foi feito para praticar **lógica de programação**, **formulários em PHP** e **estilização com CSS**.
